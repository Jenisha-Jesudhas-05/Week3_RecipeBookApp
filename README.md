# 🍴 ForkFolio - Recipe Book App

ForkFolio is a modern Android Recipe Book application built using Kotlin and RecyclerView concepts.  
The app showcases recipes with beautiful card layouts, search functionality, category filtering, and multiple RecyclerView layouts.

---

## ✨ Features

- 📋 Display recipes using RecyclerView
- ⚡ Efficient updates with ListAdapter + DiffUtil
- 🎨 Custom card-style ViewHolder UI
- 🔍 Search recipes using SearchView
- 🏷️ Filter recipes using Material Chips
- 📱 Toggle between:
  - List Layout
  - Grid Layout
- 🍔 Detailed recipe screen on item click

---

## 🛠️ Concepts Used

- Kotlin
- RecyclerView
- ListAdapter
- DiffUtil
- Custom ViewHolder
- Item Click Listeners
- GridLayoutManager
- Material Components
  - FloatingActionButton (FAB)
  - Snackbar
  - Chips
  - CardView
  - SearchView
